# 💼 Personal Portfolio Website

A modern and responsive **personal portfolio website** built using **HTML** and **CSS**. It showcases my profile, skills, education, projects, and social links with a clean dark-themed UI.

## ✨ Features
- 👤 Professional profile sidebar
- 📝 About Me section
- 💻 Skills showcase
- 🎓 Education timeline
- 📂 Portfolio projects
- 📱 Fully responsive design
- 🎨 Modern dark UI

## 🛠️ Tech Stack
- HTML5
- CSS3
- ScrollReveal (CDN)

## 📂 Project Structure
```
├── index.html
├── portfolio.css
├── Photos/
└── Linked in photo.png
```

## ▶️ How to Run
1. Clone the repository.
2. Open `index.html` in your browser.
